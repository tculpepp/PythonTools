Module 3 - Background

Project Implementation

If the project design and plan were properly created, the implementation phase of the project should go smoothly. However, few projects go according to plan and this is where foresight helps. Risk areas should already be identified and should be addressed first in the implementation so recovery of alterations in the schedule can be most easily accommodated. Sometimes changes in the requirements are necessary and sometimes additional resources must be applied. The goal is to deliver a solution that meets the high priority requirements while also meeting the project delivery dates and hopefully the budget as well.
